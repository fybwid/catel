# README

Catel is an acronym for Catatan Elektronik


## Start Development
1. Clone this project:
  ```
  git clone git@github.com:fybwid/catel.git
  ```
2. Install Gem
  ```
  bundle install
  ```
3. Prepare the Database
  ```
  rails db:setup
  ```
  > Make sure PostgreSql running `sudo service postgresql start`
4. Run the server
  ```
  rails s
  ```
